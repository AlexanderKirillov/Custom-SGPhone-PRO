Описание системных APK в прошивке

Все мы наверно задумывались, о том, что наша система захламлена программами, которые нам совершенно не нужны. 

В данной инструкции мы рассмотрим приложения из системных папок system/app и priv-app/, которые можно удалять, а какие удалять совершенно нельзя.

КРАСНЫЙ- напрямую влияет на работу системы, удалить без последствий - не получится.
ЗЕЛЕНЫЙ- не влияет на работу системы, удалить - можно (читайте описание).
ГОЛУБОЙ- "Сомнительное" приложение для удаления (не на всех аппаратах удаляется без последствий для системы), лучше такое приложение оставить.
