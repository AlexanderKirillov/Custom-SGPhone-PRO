Смена анимации загрузки

* ПРИМЕЧАНИЕ: ЕСЛИ У ВАС В ПАПКЕ SYSTEM/MEDIA НЕТ ФАЙЛА BOOTANIMATION.ZIP, А ЕСТЬ ФАЙЛ С РАЗРЕШЕНИЕМ QMG, ТО ЕГО НУЖНО ПЕРЕДЕЛАТЬ В ZIP. КАК ЭТО СДЕЛАТЬ ВЫ МОЖЕТЕ ПОСМОТРЕТЬ НИЖЕ В СООТВЕТСТВУЮЩЕЙ ИНСТРУКЦИИ!

Требования для смены анимации загрузки:
  • Наличие Root прав
  • Наличие файла анимации в формате ZIP в папке system/media либо data/local. (Если формат qmg, то читайте соответствующую инструкцию по смене формата на ZIP ниже)
Процесс замены бутанимации:
1) Скачайте бутанимацию в формате ZIP для Вашего смартфона
2) Переименуйте скачанный архив zip с бутанимацией в bootanimation.zip
3) Скачиваем любой файловый менеджер, который работает с ROOT
4) Копируем скачанную бутанимацию из п.1,2 в папку system->media или data-> local и соглашаемся на замену
5) Ставим файлу bootanimation.zip разрешения rw-r--r-- и владельца-группу root-root
6) Перезагружаемся!

Инструкция по смене формата анимации с qmg на zip

Требования:
  • Права Root-пользователя
  • Любой файловый менеджер работающий с рут правами
Сменяем формат:
1) Качаем анимацию в zip для вашего разрешения экрана
2) Кидаем её в system/media либо data/local
(Если её названия не bootanimation, то пишем bootanimation.zip )
3) Заходим в папку system/bin 
4) Делаем резервную копию файла samsungani (как вариант переименовать в samsungani.bak)
5) Удаляем файл system/bin/samsungani (если вы его не переименовали)
6) Находим файл system/bin/bootanimation и называем его samsungani
7) Удаляем два файла с форматом qmg из System/media
8) Перезагружаемся
