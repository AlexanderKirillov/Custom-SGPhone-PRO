Замена системных файлов
В данной инструкции я расскажу Вам, как изменять системные файлы Android. Эта инструкция пригодится при установке модов, твиков, дополнений для Android OS.
Требования:
- Наличие Root прав
Подготовка:
- Скачайте и установите приложение Root Exploler

1) Убедитесь, что у Вас присутствуют Root права, и установлен Root Exploler.
2) Открываем программу Root Exploler, нажимаем вверху на кнопку "Права R/W", при запросе программой прав суперпользователя- предоставляем их.
3) Собственно, теперь находим нужный системный файл, долго тапаем по нему, открывается меню. Там находим пункт "Разрешения", открываем его, и запоминаем расположение галочек. Заменяем этот системный файл на свой, и выставлем разрешения как у старого файла. Если этого не сделать- замена не сработает.

Структура файловой системы Android
Структура имеет древовидный вид. У любого дерева имеется корень, в Android он также существуют. Корень — это отправной пункт в файловой структуре, с корня система обрастает папками и файлами. Корень в Android отмечается как знак /
/system/
- находится в корне памяти устройства, содержит в себе много вложенных папок, отвечает за работу установленной операционной системы.
/system/app/ - здесь находятся системные приложения и сервисы (в новых ОС Android сервисные приложения вынесли в другую папку priv-app)
/system/bin/ и /system/xbin/ - папки содержат файлы и ссылки на исполняемые бинарные файлы.
/system/etc/ - в данной папке находятся конфигурационные файлы необходимые при загрузке ОС а также необходимые в процессе работы различных программ
/system/fonts/ - папка с системными шрифтами
/system/framework/ - папка с фрэймами Android. Это основная папка, отвечающая за работу системы.
/system/lib/ - библиотеки системных приложений и сервисов
/system/media/ - папка с системными звуками и анимацией включения
/system/priv-app/ - папка с сервисами Android (Есть только в Android 4.4+)
/system/tts/ - папка с голосовыми движками системы
/system/build.prop - конфигурационный файл с помощью которого можно изменить системные настройки
/proc/
- виртуальный раздел, содержащий информацию о ядре и его конфигурации
/mnt/
-  в данный раздел монтируются карта памяти, внутренняя память и другие виртуальные разделы (также в новых версиях Android можно встретить раздел storage)
/storage/
- в данный раздел монтируются только реальные (существующие, тоесть не виртуальные)  карты памяти и внутренняя память
/cache/
- в данном разделе находятся временные файлы, сюда скачиваются обновления системы.
/dev/
- в данном разделе содержится информация о устройствах системы и файлов
/data/
- пользовательский раздел, в котором находятся установленные приложения, личные настройки
/data/app/ - здесь находятся установленные приложения, игры
/data/app-lib/ - дополнительные библиотеки необходимые для работы определенных приложений (присутствует в новых версиях Android)
/data/dalvik-cache/ - кеш-память для работы Java машины Dalvik, которая отвечает за запуск и работу приложений.
/data/data/ - в данной папке находятся  индивидуальные настройки каждого пользовательского приложения, библиотеки, и другие файлы
/data/system/ - в данном разделе находятся настройки системных приложений
/efs/
- в этом разделе находятся файлы и папки, отвечающие за IMEI
/preload/
- в данном разделе находятся дополнительные файлы и папки, которые зеркалируются в раздел /system/
