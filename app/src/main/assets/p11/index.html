Перепрошивка через CWM/TWRP

Многие пользователи сталкивались с тем, что смартфон вдруг начинает тормозить, автоматически перезагружаться, не тянуть современные игры. При таких симптомах Мы рекомендуем перепрошить Ваш смартфон на кастомную прошивку, выпускаемую сторонними разработчиками. У нее есть и свои плюсы, т.к она может основываться на самой последней версии андроид, быть оптимизированной, иметь огромное кол-во настроек производительности и интерфейса.
В данной инструкции мы разберем установку кастомных прошивок через CWM рекавери.

Требования:
  • Установленное кастомное рекавери (См. прошлые инструкции в программе)
  • Наличие прошивки в формате ZIP
Прошиваем...
1) Скачиваем кастомную прошивку в формате ZIP, и закидываем ее на телефон
2) Перезагрузимся и войдем в кастомное рекавери при помощи следующей комбинации клавиш: Громкость вверх+ кнопка домой+ кнопка питания
3) В рекавери сделать (На примере CWM):
Wipe data/factory reset - не нужно делать при обновлении версии одной прошивки
Wipe Cache partition
Advanced => Wipe Dalvik Cache
Mounts and Storage => Format System
4) В рекавери в главном меню нажать Install zip  (Или Install zip from sdcard)
5) Выбираем тип памяти, откуда рекавери будет считывать нашу скачанную прошивку (Внутренняя (Internal) или внешняя (External)- накопитель или карта памяти, в зависимости от того, куда копировали файл прошивки)
6)  Найти файл прошивки, нажать на него и согласиться на установку.
7)  Перезагрузится.
